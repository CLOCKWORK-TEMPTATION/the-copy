## جدول مشكلات الربط (Wiring Issues)

### 1. مسارات الاستيراد المكسورة - ملفات AI Flows

**النوع:** Import Path مكسور  
**الملفات المتأثرة:**
- `frontend/src/ai/dev.ts` (السطر 4-5)

**المشكلة:**
```typescript
// frontend/src/ai/dev.ts
import "@/ai/flows/initial-dramatic-analysis.ts";  // ❌ الملف غير موجود
import "@/ai/flows/seven-layer-dramatic-analysis.ts"; // ❌ الملف غير موجود
```

**السبب:** المسارات تشير إلى ملفات غير موجودة في المشروع. المجلد `frontend/src/ai/flows/` غير موجود.

**الإصلاح المقترح:**
1. إنشاء الملفات المفقودة في `frontend/src/ai/flows/`، أو
2. حذف هذه الاستيرادات إذا لم تعد مطلوبة، أو
3. تحديث المسارات لتشير إلى الملفات الصحيحة

---

### 2. مسارات Alias غير متطابقة - @core/enums و @core/types

**النوع:** Alias خاطئ أو غير متطابق  
**الملفات المتأثرة:**
- جميع الملفات التي تستورد من `@core/enums` و `@core/types`

**المشكلة:**
في `frontend/tsconfig.json`:
```json
"@core/*": ["./src/lib/drama-analyst/*"]
```

لكن الملفات الفعلية موجودة في:
- `@core/enums` → `./src/lib/drama-analyst/enums.ts` ✅ (صحيح)
- `@core/types` → `./src/lib/drama-analyst/core/types.ts` ⚠️ (يحتاج إلى `@core/core/types`)

**الملفات التي تستخدم هذه المسارات:**
- `frontend/src/lib/drama-analyst/orchestration/agentFactory.ts`
- `frontend/src/lib/drama-analyst/orchestration/orchestration.ts`
- `frontend/src/lib/drama-analyst/services/backendService.ts`
- وغيرها...

**الإصلاح المقترح:**
1. **الخيار 1:** نقل `core/types.ts` إلى `drama-analyst/types.ts` مباشرة
2. **الخيار 2:** تحديث الاستيرادات لتستخدم `@core/core/types`
3. **الخيار 3:** إضافة alias جديد في tsconfig:
   ```json
   "@core/types": ["./src/lib/drama-analyst/core/types.ts"]
   ```
